## Installation de dependecias locales

Instalar las siguientes dependencias dentro del proyecto

```bash
 npm init -y
 npm i nodemon
 npm i express
 npm i dotenv
 npm i express-validator
 npm i mongoose
 npm i bcryptjs
 npm i jsonwebtoken
 npm i cors
 npm i moment
```

## Installation de dependecias globales

Instalar las siguientes dependencias
  
```bash
  npm i typescript -g
  npm i ts-node -g
  npm i nodemon -g
```
